# 👤 Personal Profile Card

A responsive personal profile card built using **HTML5** and **CSS3**. The project showcases a clean and modern user interface with a soft gradient background, profile image, hover animations, and social media icons.

## 🚀 Live Demo
https://salmaashourabdelaziem.github.io/Personal-Card/

## 📋 Features

- Responsive profile card layout
- Modern and minimal UI design
- Gradient background
- Hover animation on the profile card
- Social media icons using Remix Icons
- Clean typography with Google Fonts
- Fully responsive across different screen sizes

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Google Fonts
- Remix Icons
